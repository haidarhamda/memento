## Cara Bermain
Control: \
Kiri: A \
Kanan: D \
Jump: space 

![cherry](./doc/cherry-1.png)  
Jika player mendapatkan cherry, dapat 3 poin  
  
![gem](./doc/gem-1.png)  
Jika player mendapatkan gem, dapat 5 poin  
  
![block](./doc/block.png)  
Jika player mendapatkan block, dapat -4 poin
